# Landing Page Profissional — Amanda Campos

Projeto desenvolvido em HTML, CSS e JavaScript puro, sem Bootstrap, Tailwind ou frameworks externos.

## Estrutura

```text
landing-amanda/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
└── assets/
    └── avatar.png
```

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.

## Como trocar a foto

Substitua o arquivo `assets/avatar.png` por uma foto própria com o mesmo nome, ou altere o caminho da imagem no `index.html`.

## Como alterar links dos projetos

No `index.html`, procure pelos cards dentro da seção `#projetos` e substitua `href="#"` pelos links do GitHub, deploy ou portfólio.

## Recursos implementados

- Layout responsivo para desktop, tablet e mobile;
- Navbar sticky com menu hambúrguer no mobile;
- Rolagem suave;
- Destaque automático do link ativo no menu;
- Cards com hover;
- Animação suave de entrada;
- Botão de voltar ao topo;
- Estrutura semântica com `nav`, `header`, `main`, `section`, `article`, `aside` e `footer`.
